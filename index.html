I have been fascinated with technology ever since I got my first iPod Touch back in the second grade. I was always curious as to how I was able to use a singular machine to play games, watch movies, and talk to friends, wherever I was.

Fast-forward ten years to when I was 17, just gradutating high school, and still just as curious as I was when I was a child. It was at this point that I built my first PC. When I got it all set up, I immediately went to Google to start to learn to code, primarily using SoloLearn beginner courses. I worked through the Python course, while building some (admittedly bad) projects.

I started studying at the University of Victoria during the fall of 2020, right in the swing of COVID19. While it wasn't exactly what I expected for my first year of university, I still had a great time learning more about Python and Java while refining my problem-solving skills. During the first semester, I made my first larger project using Python, that was a Craps game simulator played in the command line.

So far in my second year I have been having a lot of fun. I have really enjoyed being in Victoria and being able to explore the campus and the city. Even the ten minute walk between classes is a great way to refresh my mind after a lecture or a lab. I have had the opportunity to use new technologies this year, including HTML, CSS, JavaScript and C. These technologies have been so much fun to learn because they really opened my eyes to entirely new methods of programming. For example, HTML and CSS are written completely differently than Python and Java, and they hold the power to precisely change what the user sees when looking at the final version of the project.

C, on the other hand, was a completely different beast. Because it is such a low-level language, I got the opportunity to immerse myself in pointers. It was challenging at first but once I got it I came to appreciate the beauty of using memory locations rather than memory data. I also got my first opportunity to really learn Git, rather than just upload files after the project was finished. I am now comfortable working on projects on both my laptop and desktop, and reverting back to older versions if I decide I don't like any changes I've made.
